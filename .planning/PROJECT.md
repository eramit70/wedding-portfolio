# Project: Deepak & Reshami Wedding Portfolio

## Goal
Create a premium, single-page wedding portfolio website with a public-facing page and an admin dashboard. No backend or DB support (use `localStorage`).

## Context
A wedding celebration for Deepak and Reshami. The user wants to host this easily without backend/DB complexity.

## Technology Stack
- HTML5 / Vanilla JavaScript (ES6+)
- Vanilla CSS (Premium Aesthetics: Inter/Roboto fonts, gradients, glassmorphism, animations)
- Data Persistence: `localStorage`
